# MailStats

MailStats is a simple dashboard that shows what's happening with the emails
we send out — like our student portal notifications, newsletters, and
system alerts.

## What it does

Every time an email is sent, something can happen to it: it gets delivered
successfully, it bounces back because the address doesn't exist, it gets
opened by the recipient, a link inside gets clicked, or someone unsubscribes.

MailStats collects all of that information automatically and shows it in
one place, so we can answer questions like:

- Are our emails actually reaching people?
- Which addresses keep bouncing?
- Is a particular domain having delivery problems?
- How many people are opening or clicking our emails?

## How it's organized

Each domain we send email from (for example, our student portal or a
specific mail agent) gets tracked separately, with its own private link and
its own set of stats. This keeps things organized when we're sending from
more than one system.

## Who this is for

This is an internal tool for keeping an eye on our email delivery health.
It's password-protected, so only people with access can view the dashboard.
